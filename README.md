# UnsplashProject
An app that allows to view, search, download high-quality photos using Unsplash API
   Coroutines, Retrofit, Room, Koin, Paging
## Applied concepts
* Arch. patterns: MVVM, Clean Architecture 
* DI: Koin
* Paging
* Network: Retrofit
* Multithreadin: Coroutines
* DB: Room
